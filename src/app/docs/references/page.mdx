# References

A collection of libraries, tools, and projects that inspired and supported the creation of Prisma UI.

## Inspiration

### Enhanced Button

The first component built for this library was inspired by the **Enhanced Button** project.
It served as the starting point and the conceptual foundation for what this library aims to become.

- [enhanced-button.vercel.app](https://enhanced-button.vercel.app/)

---

## Tools & Infrastructure

### Nextra

The documentation site is powered by **Nextra**, a Next.js-based static site framework built for docs.

- [nextra.site/docs](https://nextra.site/docs)

### shadcn/ui Blocks

The landing page was built using **shadcnblocks**, a collection of pre-built sections and layouts based on shadcn/ui.

- [shadcnblocks.com](https://www.shadcnblocks.com/)

---

## Component Libraries

The components in Prisma UI draw visual and behavioral inspiration from the following open-source libraries:

### shadcn/ui

The foundation of Prisma UI's component system. Provides accessible, unstyled primitives built on Radix UI and Tailwind CSS.

- [ui.shadcn.com](https://ui.shadcn.com/)

### Magic UI

A collection of animated React components built with Tailwind CSS and Framer Motion.

- [magicui.design](https://magicui.design/)

### Kokonut UI

A modern component library focused on beautiful, ready-to-use UI blocks.

- [kokonutui.com](https://kokonutui.com/)
