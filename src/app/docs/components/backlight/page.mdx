# Backlight

An SVG-based glow effect that creates a saturated blur behind images, videos, and SVGs. Inspired by [Magic UI's Backlight](https://magicui.design/docs/components/backlight).

## Installation

<InstallSnippet registryUrl='https://prismaui.com/components/backlight.json' />

## Import

```tsx
import { Backlight } from '@/components/ui/backlight';
```

---

## Image Glow

Wrap any image to create a vivid glow effect behind it.

<ComponentPreview>
  <Backlight className='rounded-lg'>
    <img
      src='https://picsum.photos/320/200'
      alt='Backlight demo'
      className='rounded-lg'
      width={320}
      height={200}
    />
  </Backlight>
</ComponentPreview>

```tsx
<Backlight className='rounded-lg'>
  <img
    src='https://picsum.photos/320/200'
    alt='Example'
    className='rounded-lg'
    width={320}
    height={200}
  />
</Backlight>
```

---

## Custom Blur

Adjust the `blur` prop to control the glow intensity.

<ComponentPreview>
  <Backlight blur={40} className='rounded-lg'>
    <img
      src='https://picsum.photos/322/204'
      alt='Strong glow'
      className='rounded-lg'
      width={322}
      height={204}
    />
  </Backlight>
</ComponentPreview>

```tsx
<Backlight blur={40} className='rounded-lg'>
  <img
    src='https://picsum.photos/322/204'
    alt='Strong glow'
    className='rounded-lg'
    width={322}
    height={204}
  />
</Backlight>
```

---

## Props

| Prop        | Type           | Default | Description                                         |
| ----------- | -------------- | ------- | --------------------------------------------------- |
| `children`  | `ReactElement` | —       | The image, video, or SVG to apply the backlight to. |
| `className` | `string`       | —       | Additional CSS classes for the wrapper.             |
| `blur`      | `number`       | `20`    | Blur intensity of the glow.                         |
