# Infinite Slider

An infinite scrolling slider that smoothly loops through its children. Inspired by [Motion Primitives' Infinite Slider](https://motion-primitives.com/docs/infinite-slider). Supports horizontal and vertical directions with customizable speed.

## Installation

<InstallSnippet registryUrl='https://prismaui.com/components/infinite-slider.json' />

## Import

```tsx
import { InfiniteSlider } from '@/components/ui/infinite-slider';
```

## Props

| Prop           | Type                         | Default        | Description                  |
| -------------- | ---------------------------- | -------------- | ---------------------------- |
| `children`     | `React.ReactNode`            | —              | Slider content.              |
| `gap`          | `number`                     | `16`           | Gap between items in pixels. |
| `speed`        | `number`                     | `100`          | Speed in pixels per second.  |
| `speedOnHover` | `number`                     | —              | Speed when hovered.          |
| `direction`    | `'horizontal' \| 'vertical'` | `'horizontal'` | Scroll direction.            |
| `reverse`      | `boolean`                    | `false`        | Reverse direction.           |
| `className`    | `string`                     | —              | Additional CSS classes.      |
