# Tilt Card

A 3D tilt wrapper that applies a perspective tilt effect on mouse movement. Inspired by [Motion Primitives' Tilt](https://motion-primitives.com/docs/tilt). Wrap any Card or element for interactive depth.

## Installation

<InstallSnippet registryUrl='https://prismaui.com/components/tilt.json' />

## Import

```tsx
import { Tilt } from '@/components/ui/tilt';
```

## Props

| Prop             | Type              | Default | Description                    |
| ---------------- | ----------------- | ------- | ------------------------------ |
| `children`       | `React.ReactNode` | —       | Content to tilt.               |
| `className`      | `string`          | —       | Additional CSS classes.        |
| `style`          | `MotionStyle`     | —       | Motion style overrides.        |
| `rotationFactor` | `number`          | `15`    | Max rotation angle in degrees. |
| `isReverse`      | `boolean`         | `false` | Reverse the tilt direction.    |
| `springOptions`  | `SpringOptions`   | —       | Spring animation config.       |
