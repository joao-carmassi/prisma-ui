# Glow Effect

A customizable animated glow effect with multiple animation modes. Inspired by [Motion Primitives' Glow Effect](https://motion-primitives.com/docs/glow-effect). Drop inside any element with `relative overflow-hidden`.

## Installation

<InstallSnippet registryUrl='https://prismaui.com/components/glow-effect.json' />

## Import

```tsx
import { GlowEffect } from '@/components/ui/glow-effect';
```

---

## Card Glow

Place `GlowEffect` as a **sibling** before the card content inside a `relative` wrapper (no `overflow-hidden` on the parent). The card needs `relative z-10` and an opaque background so the glow glows around it, not through it.

<ComponentPreview>
  <div className='relative'>
    <GlowEffect
      colors={['#FF5733', '#33FF57', '#3357FF', '#F1C40F']}
      mode='rotate'
      blur='medium'
    />
    <Card className='relative z-10 w-85'>
      <CardHeader>
        <CardTitle>Glow Effect</CardTitle>
        <CardDescription>
          Rotating gradient glow around the card.
        </CardDescription>
      </CardHeader>
      <CardContent>
        <p className='text-sm text-muted-foreground'>
          The glow sits behind the card as a sibling, creating a halo effect at
          the edges.
        </p>
      </CardContent>
    </Card>
  </div>
</ComponentPreview>

```tsx
<div className='relative'>
  <GlowEffect
    colors={['#FF5733', '#33FF57', '#3357FF', '#F1C40F']}
    mode='rotate'
    blur='medium'
  />
  <Card className='relative z-10'>
    <CardHeader>
      <CardTitle>Glow Effect</CardTitle>
      <CardDescription>Rotating gradient glow around the card.</CardDescription>
    </CardHeader>
    <CardContent>
      <p className='text-sm text-muted-foreground'>Content goes here.</p>
    </CardContent>
  </Card>
</div>
```

## Button Glow

Works the same with any inline element.

  <div className='relative inline-flex'>
    <GlowEffect
      colors={['#A97CF8', '#F38CB8', '#FDCC92']}
      mode='rotate'
      blur='soft'
    />
    <Button className='relative z-10'>Glowing Button</Button>
  </div>
</ComponentPreview>

```tsx
<div className='relative inline-flex'>
  <GlowEffect
    colors={['#A97CF8', '#F38CB8', '#FDCC92']}
    mode='rotate'
    blur='soft'
  />
  <Button className='relative z-10'>Glowing Button</Button>
</div>
```

## Animation Modes

Use the `mode` prop to change the glow animation: `rotate`, `pulse`, `breathe`, `colorShift`, `flowHorizontal`, or `static`.

      pulse
    </span>
  </div>
  <div className='relative h-20 w-32 overflow-hidden rounded-lg bg-card'>
    <GlowEffect mode='breathe' />
    <span className='relative flex h-full items-center justify-center text-sm font-medium'>
      breathe
    </span>
  </div>
  <div className='relative h-20 w-32 overflow-hidden rounded-lg bg-card'>

      flowHorizontal

  <div className='relative h-20 w-32 overflow-hidden rounded-lg bg-card'>

## Props

| Prop         | Type                                                                                           | Default                                        | Description                    |
| ------------ | ---------------------------------------------------------------------------------------------- | ---------------------------------------------- | ------------------------------ |
| `colors`     | `string[]`                                                                                     | `['#FF5733', '#33FF57', '#3357FF', '#F1C40F']` | Glow colors.                   |
| `mode`       | `'rotate' \| 'pulse' \| 'breathe' \| 'colorShift' \| 'flowHorizontal' \| 'static'`             | `'rotate'`                                     | Animation mode.                |
| `blur`       | `number \| 'softest' \| 'soft' \| 'medium' \| 'strong' \| 'stronger' \| 'strongest' \| 'none'` | `'medium'`                                     | Blur intensity.                |
| `transition` | `Transition`                                                                                   | —                                              | Motion transition override.    |
| `scale`      | `number`                                                                                       | `1`                                            | Scale of the glow.             |
| `duration`   | `number`                                                                                       | `5`                                            | Duration per cycle in seconds. |
| `className`  | `string`                                                                                       | —                                              | Additional CSS classes.        |
| `style`      | `React.CSSProperties`                                                                          | —                                              | Inline styles.                 |
