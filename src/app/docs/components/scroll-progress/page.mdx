# Scroll Progress

An animated scroll progress bar fixed to the top of the page. Inspired by [Magic UI's Scroll Progress](https://magicui.design/docs/components/scroll-progress). The gradient line scales with the user's scroll position.

## Installation

<InstallSnippet registryUrl='https://prismaui.com/components/scroll-progress.json' />

## Import

```tsx
import { ScrollProgress } from '@/components/ui/scroll-progress';
```

---

## Usage

Place the component at the root of your layout or page:

<ComponentPreview>
  <div className='relative h-2 w-full overflow-hidden rounded-full bg-muted'>
    <div className='h-full w-3/5 rounded-full bg-linear-to-r from-[#A97CF8] via-[#F38CB8] to-[#FDCC92]' />
  </div>
</ComponentPreview>

```tsx
export default function Layout({ children }) {
  return (
    <>
      <ScrollProgress />
      {children}
    </>
  );
}
```

---

## Custom Colors

  </div>
</ComponentPreview>

```tsx
<ScrollProgress className='bg-linear-to-r from-blue-500 via-purple-500 to-pink-500' />
```
